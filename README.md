# Pomodoro
### 프로젝트 설명
플러터로 구현한 간단한 포모도로 앱입니다.

### 기능
#### 시작 버튼 클릭
1. 25분 카운트다운이 시작됩니다.
2. 아이콘이 일시정지 버튼으로 바뀝니다.
3. 25분이 지나면 Pomodoro Count가 1씩 증가합니다.

#### 중지 버튼 클릭
1. 25분 카운트다운이 일시정지 합니다.
2. 아이콘이 시작 버튼으로 바뀝니다.
